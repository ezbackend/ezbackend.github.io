# EzBackend

> Build robust, scalable backends without the boilerplate.

EzBackend is a backend framework that lets you focus on your product, not the setup. Get from zero to a fully functional REST API in minutes.

## Quick Start

```bash
npm install -g ezbackend
ezbackend create my-project
cd my-project
ezbackend start
```

## Features

- ⚡ **Lightning Fast Setup** — Minimal boilerplate, maximum productivity
- 🗄 **Database Agnostic** — Works with your favorite databases
- 🛡 **Secure by Default** — Built-in security best practices
